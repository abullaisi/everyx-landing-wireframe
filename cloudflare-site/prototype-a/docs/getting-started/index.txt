## What is Scalar?

Scalar is a modern headless CMS that provides a powerful admin interface, flexible content modeling, and developer-friendly APIs. It's designed to scale from simple websites to complex applications.

  ### Flexible Content Models

  ### Auto-generated APIs

## 5-Minute Setup

    ### Install Scalar ```bash npx create-scalar-app my-app cd my-app npm
    install ```

    ### Configure your database ```bash title=".env"
    DATABASE_URL="postgresql://username:password@localhost:5432/mydb"
    SCALAR_SECRET="your-secret-key" ```

    ### Start development ```bash npm run dev ``` Visit
    `http://localhost:3000/admin` to access the admin panel.

## Your First Content Model

Define a simple blog post model:

```typescript title="schema/post.ts"

export const post = defineModel({
  name: 'post',
  label: 'Blog Post',

  fields: {
    title: {
      type: 'text',
      required: true,
      label: 'Title',
    },

    content: {
      type: 'richtext',
      required: true,
      label: 'Content',
    },

    publishedAt: {
      type: 'datetime',
      label: 'Published At',
      defaultValue: () => new Date(),
    },
  },
});
```

## Fetch Content

Query your content using the generated API:

  ### REST API

    ```bash
    curl https://your-app.com/api/post
    ```

  ### GraphQL

    ```graphql
    query {
      posts {
        id
        title
        content
        publishedAt
      }
    }
    ```

  ### JavaScript SDK

    ```javascript
    import { scalar } from '@/lib/scalar';
    
    const posts = await scalar.content.findMany({
      model: 'post',
      sort: '-publishedAt'
    });
    ```
